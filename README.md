# CitaFácil 📅

> SaaS de reservaciones de citas para negocios locales — salones, médicos, dentistas y más.

## ✨ Características

- 🏢 **Multi-negocio** — Cada negocio tiene su propia página de reservaciones
- 📅 **Gestión de horarios** — Configura disponibilidad por día y hora
- 💳 **Planes Gratis / Premium** — Integración con Stripe
- 📧 **Recordatorios por email** — Solo plan Premium
- 📊 **Estadísticas** — Solo plan Premium
- 🌙 **Dark mode** — Diseño profesional

## 🛠 Stack Técnico

| Tecnología | Uso |
|-----------|-----|
| Next.js 14 (App Router) | Frontend + Backend (API Routes) |
| Supabase | Base de datos PostgreSQL + Autenticación |
| Stripe | Pagos de suscripción mensual |
| Tailwind CSS | Estilos dark mode |
| Vercel | Deploy en producción |

## 🚀 Instalación local

```bash
git clone https://github.com/TU_USUARIO/citafacil.git
cd citafacil
npm install
cp .env.example .env.local
npm run dev
```
